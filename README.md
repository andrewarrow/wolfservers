# wolfservers
go cli to help create various crypto systems at places like aws,
digitalocean,vultr,linode,google,and more.

# example

./wolfservers ed255

./wolfservers sqlite

./wolfservers fresh2linode --sure

./wolfservers update-ips --producer=ip1 --relay=ip2 --name=wolf-ABCD

./wolfservers ls
